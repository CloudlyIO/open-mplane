Detailed Description

U-Plane configuration functions - O-RAN WG4 M-Plane Spec Chapter 15.2.

The user plane (U-Plane) is part of the Open Fronthaul interface between the O-RU and the O-DU. This module allows for U-Plane configuration on the O-RU via the M-Plane.
